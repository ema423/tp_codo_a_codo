Y te preguntarás...¿Que lenguajes debo aprender? ¿Por donde empezar? Aqui debajo hay algunos de los más utilizados:

HTML


El HTML ( HyperText Markup Language o lenguaje de marcado de hipertexto en español) es el lenguaje que, mediante el uso de etiquetas, permite armar una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web (con texto, imágenes, videos, etc.).


CSS


El lenguaje CSS (Cascading Style Sheets o Hojas de estilo en cascada en español) es un lenguaje que sirve para personalizar los estilos de un sitio web.

La principal ventaja que brinda es la posibilidad de definir el diseño de una web sin tener que modificar su contenido.


JavaScript


Este lenguaje se utiliza principalmente del lado del cliente, permitiendo mejoras en la interfaz de usuario y páginas web más dinámicas.

Una de las principales ventajas que tiene este lenguajes es que el código JavaScript puede ejecutarse localmente en el navegador del usuario, logrando que este puede responder a las acciones del usuario con rapidez.

Por otra parte, el código JavaScript puede detectar acciones de los usuarios que HTML por sí sola no puede, como pulsaciones de teclado.


Ruby


Con Ruby se pueden crear muchisimas cosas , desde desarrollar aplicaciones hasta procesar datos en Back-end.

Su principal ventaja es la simplicidad de su sintaxis, que es bastante fácil de entender.


Java


Es un lenguaje de programación orientado a objetos, que está presente en muchisimos sitios web, videojuegos y aplicaciones.


Python


Es uno de los lenguajes más simples, intuitivos y a su vez completos dentro de la programación.Esto lo hace ideal para quienes estén iniciandose en este mundo.


PHP


PHP es un lenguaje de programación destinado a desarrollar aplicaciones para la web y crear páginas web, favoreciendo la conexión entre los servidores y la interfaz de usuario.


SQL


Es un lenguaje que sirve para acceder y administrar bases de datos.
